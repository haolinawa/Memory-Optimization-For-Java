# Memory-Optimization-For-Java

## 什么是 MOFJ?
MOFJ 又名摸福建（ ，可以优化java程序（minecraft）的内存使用过多而泄露问题（是这样嘛 ~ ）

原理就是把程序不使用内存先移到虚拟内存中以空闲更多的空间，程序需要使用内存时移出来cwc。

##  我是小白 我该如何使用程序？
so easy 你只需要下载仓库内的MOFJ.exe或者按右边的"Release"下载MOFJ.exe来运行就行了 但是这个程序没钱买数字签名 所以会360容易误报（ 

如果你不相信咱编译好的文件的话 你可以自己编译一个版本出来哦。

## 我是开发者 我需要源码 但是在哪里呢？
请下载仓库内的MOFJ.e和模块文件夹内的全部模块，然后使用易语言程序打开后缀为.e的文件就好呀awa

模块是引用别人的哦 所以模块是没有源码文件滴

## 真的对java（MC）有效吗？
对于大型整合包和低配置机器比较有效，如果你的内存非常充裕，这个模组将很难起到效果。

## 为什么我清理内存的时候老是卡？
1.请把内存清理间隔建议设置为300s以上 或者 升级你的配置

2.如果清理时候会卡顿，建议添加 JVM 参数“-XX:+ExplicitGCInvokesConcurrent”来解决清理时候的卡顿问题。
